Daily reminder cron — staging

A scheduled job that runs every minute and delivers the daily journal reminder to signed-in users at the time each one picked, in their own timezone. (Guests receive their reminder locally on the device, so they aren't sent from here.) It only pushes to a user at their chosen minute, so most runs send nothing. When switched OFF, the job stays scheduled but makes no calls at all — no notifications, no FCM usage. Staging is normally kept OFF; production should be ON.

Enter the admin secret, then refresh to see the current state.
